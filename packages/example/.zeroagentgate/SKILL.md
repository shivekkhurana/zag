# Todo Service

A simple todo management service

## Installation

```bash
# Install zag CLI
bun install -g @ai26/zag

# Register with the service
zag setup http://localhost:8000
```

## Usage

After registration, execute actions:

```bash
zag exec http://localhost:8000 <action-id> [--data '{"key": "value"}']
```

## API Reference

See the [manifest](manifest.json) for available actions and schemas.

### Available Actions

- `list-todos`: GET /api/todos - List all todos
- `create-todo`: POST /api/todos - Create a new todo
- `delete-todo`: DELETE /api/todos/:id - Delete a todo by ID
